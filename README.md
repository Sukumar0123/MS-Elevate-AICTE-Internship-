# MS-Elevate-AICTE-Internship


Student Performance Analysis Dashboard


Project Overview

The Student Performance Analysis Dashboard is a data visualization project that analyzes student academic performance using interactive charts and graphs.

The dashboard converts raw student data into meaningful insights that help educators understand student performance trends and make better academic decisions.

Objectives

Analyze student performance using data visualization

Identify patterns and trends in student scores

Compare performance across different categories

Provide insights for academic improvement

Dataset

The dataset contains student academic information such as:

Gender

Ethnicity

Parental Level of Education

Test Preparation Course

Math Score

Reading Score

Writing Score

Technology Stack
Technology	Purpose
Excel	Data Cleaning and Preparation
Power BI	Dashboard Visualization
Dashboard Features

Student count overview

Average Math, Reading, and Writing scores

Gender-based performance comparison

Ethnicity analysis

Parental education impact

Test preparation impact
